# GridWar

Progetto universitario per il corso di Modellazione e Gestione della Conoscenza
Università di Camerino — A.A. 2025/2026

GridWar è un gioco di ruolo tattico a turni ispirato alla serie Fire Emblem. Il giocatore comanda un gruppo di unità su una mappa a griglia, affrontando le forze nemiche in combattimenti a turni. Ogni unità appartiene a una classe (Guerriero, Mago, Arciere, Cavaliere, Ladro) con statistiche e armi proprie, e i tipi di arma interagiscono tramite il triangolo delle armi (Spada > Ascia > Lancia > Spada).
